# Internal Documentation

This directory contains internal planning, operational, and sales documents that are not part of the public platform documentation.

These files are present in this repository for version control purposes only. They are not intended for public review.

**For public platform documentation, see [docs/](../docs/).**

## Contents

| Directory   | Contents                                                                              |
| ----------- | ------------------------------------------------------------------------------------- |
| `internal/` | Sales artifacts, readiness audits, pricing, licensing playbooks, operational runbooks |
| `policies/` | Internal enrollment, funding, and staff policies                                      |
| `outreach/` | Employer agreement templates, sponsorship pitches, referral materials                 |

## Note

If this repository is ever made fully public or transferred, this directory should be removed or moved to a private repository before publication.
