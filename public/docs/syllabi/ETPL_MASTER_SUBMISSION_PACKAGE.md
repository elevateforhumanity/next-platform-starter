## Indiana DWD/WorkOne Partnership

### WorkOne Referral Process

1. **Eligibility Determination:** WorkOne counselor assesses WIOA/WRG eligibility
2. **Program Selection:** Student selects ETPL-approved program
3. **Enrollment:** Elevate for Humanity processes enrollment
4. **Training:** Student completes program with support services
5. **Job Placement:** Career services assist with employment
6. **Follow-Up:** 12-month tracking for outcomes reporting
