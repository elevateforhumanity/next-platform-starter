## Hour Requirements (Minimums)

| Category                         | IDOH Minimum    | Program Target | Notes                      |
| -------------------------------- | --------------- | -------------- | -------------------------- |
| Classroom / online-equivalent    | 30 hrs          | 30 hrs         | LMS-delivered didactic     |
| Clinical — lab setting           | up to 21 hrs    | 21 hrs         | Skills lab at CMI facility |
| Clinical — licensed LTC facility | at least 24 hrs | 24 hrs         | CMI clinical site          |
| **Total**                        | **75 hrs**      | **75 hrs**     | Minimum; may exceed        |

**Completion rule:** A student is not complete until ALL three hour buckets are satisfied and signed off. The LMS must not issue a completion certificate based on didactic alone.

## Resident Care Procedures (RCPs)

IDOH Appendix A defines 72 Resident Care Procedures. These are skills, not lessons — they are practiced and signed off during lab (lessons 26–28) and verified during LTC clinical (lessons 29–30).

The LMS must track RCP sign-off separately from lesson completion. A student who completes all 25 didactic lessons but has not had all required RCPs signed off by an instructor is **not complete**.

RCP tracking is stored in `step_submissions` with `step_type = 'lab'` and requires instructor sign-off (`instructor_approved = true`).

The full RCP list (procedures 1–72) is sourced from:  
https://www.in.gov/health/ltc/files/RCPS-092624.pdf

## Completion Gate Logic

A student reaches program completion only when ALL of the following are true:

1. **All 25 didactic lessons passed** — each quiz ≥ 70%
2. **All 3 lab lessons signed off** — instructor approval on `step_submissions` for cna-l26, cna-l27, cna-l28
3. **Both LTC clinical lessons signed off** — instructor approval on `step_submissions` for cna-l29, cna-l30
4. **Minimum hours met** — `training_enrollments.hours_completed ≥ 75` with `clinical_hours_completed ≥ 45` (requires a separate hours tracking column or a `cmi_attendance` aggregate)
5. **Comprehensive final exam passed** — a separate `step_type = 'exam'` lesson covering all 30 IDOH lessons, pass threshold ≥ 70%

The LMS **must not** issue a completion certificate until all five conditions are satisfied. The current `program_completion_certificates` trigger must be updated to enforce this gate.

## Admin Documentation Requirements (NATCEP)

These are program-level operational requirements, not LMS features. They must exist as physical or digital records maintained by the program director.

| Requirement                                      | Source            | Status                            |
| ------------------------------------------------ | ----------------- | --------------------------------- |
| State Form 629 (Application to Operate)          | IDOH              | Must be filed before any training |
| TTT certificates for all instructors             | Admin Standard #1 | Must be on file per instructor    |
| Current RN license copies for all instructors    | Admin Standard #1 | Must be on file                   |
| Clinical site agreement (CMI ↔ LTC facility)     | IDOH requirement  | Must be executed before clinical  |
| Classroom/lab photos (2–3 each)                  | IDOH requirement  | Submitted with Form 629           |
| Master Training Schedule                         | IDOH form         | Per cohort                        |
| Training Supply and Equipment Form               | IDOH form         | Per cohort                        |
| Student time records (dates, topics, signatures) | IDOH requirement  | Per student, per session          |
| Certificate of completion template               | IDOH requirement  | Must state "minimum of 75-hour"   |
| Student pre-admission assessment (math/English)  | IDOH requirement  | Per student                       |
| Comprehensive final exam                         | IDOH requirement  | Aligned to all 30 lessons         |
| Student file checklist                           | IDOH requirement  | Per student                       |

The LMS can generate or store some of these (time records, final exam, completion certificate) but cannot substitute for the physical program-level documentation.

## What This Matrix Does Not Cover

- **NATCEP program approval status** — this matrix describes the curriculum structure. It does not mean the program is approved. Approval requires filing Form 629 and receiving written authorization from IDOH before any training begins.
- **Competency evaluation (CE)** — the state exam (written + skills) is administered by a third-party evaluator, not by this LMS. The LMS completion certificate is a training completion record, not a certification.
- **Registry listing** — CNA certification and registry listing are handled by IDOH after the student passes the state competency evaluation. The LMS has no role in that process.
- **RCP content** — the 72 RCP procedures are not yet mapped to individual `step_submissions` rows. That mapping must be done before the lab lessons go live.
