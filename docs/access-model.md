# Access Model

Elevate for Humanity operates the Workforce Operating System as a managed platform. Access to the live system is provided under formal agreements — not through this public repository.

## Access Tiers

### 1. Managed Platform Access

The standard access model for organizations that want to use the platform without managing infrastructure.

| Item                   | Details                                                                        |
| ---------------------- | ------------------------------------------------------------------------------ |
| **What you get**       | Full platform access — LMS, enrollment, compliance, reporting, employer portal |
| **What you don't get** | Source code, database access, infrastructure control                           |
| **Operated by**        | Elevate for Humanity — we run it, maintain it, update it                       |
| **Setup fee**          | $7,500–$15,000 (one-time)                                                      |
| **Monthly**            | $1,500–$3,500 depending on scope                                               |
| **Non-payment**        | Automatic suspension                                                           |

Suitable for: workforce boards, community colleges, training providers, employer consortia.

### 2. Enterprise Source-Use License

For qualified enterprises that need to deploy and operate the platform independently.

| Item                   | Details                                                                      |
| ---------------------- | ---------------------------------------------------------------------------- |
| **What you get**       | Restricted source-use rights, deployment support, documentation              |
| **What you don't get** | Ownership of IP, right to sublicense, right to white-label without agreement |
| **Minimum**            | $75,000                                                                      |
| **Requirements**       | Formal agreement, use-case approval, compliance alignment                    |

Suitable for: state agencies, large workforce systems, enterprise training operators.

### 3. Public Transparency (This Repository)

This repository is public for transparency, evaluation, and documentation purposes.

| Item                   | Details                                                                          |
| ---------------------- | -------------------------------------------------------------------------------- |
| **What you can see**   | Application structure, API design, compliance documentation, architecture        |
| **What is not here**   | Production secrets, live student data, full video library, proprietary AI assets |
| **What you can do**    | File issues, review code, evaluate the platform for procurement                  |
| **What you cannot do** | Deploy this as a production system, use code commercially without a license      |

See [docs/repository-scope.md](repository-scope.md) for the full list of what is and is not included.
