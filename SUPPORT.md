## Technical / Repository Issues

File a GitHub issue using the appropriate template:

- **[Bug Report](https://github.com/elevateforhumanity/Elevate-lms/issues/new?template=bug_report.yml)** — defects in the platform
- **[Feature Request](https://github.com/elevateforhumanity/Elevate-lms/issues/new?template=feature_request.yml)** — new capabilities or improvements
- **[Compliance / Data Concern](https://github.com/elevateforhumanity/Elevate-lms/issues/new?template=compliance_concern.yml)** — data handling or regulatory concerns

## Security Vulnerabilities

Do not file public issues for security vulnerabilities. See [docs/SECURITY.md](docs/SECURITY.md) for the private disclosure process.
