## SKILL 1: HAND HYGIENE (SOAP AND WATER)

_This skill is evaluated on every competency exam._

| Step                                                               | S   | U   |
| ------------------------------------------------------------------ | --- | --- |
| 1. Turns on water to comfortable temperature                       |     |     |
| 2. Wets hands under running water                                  |     |     |
| 3. Applies soap to cover all hand surfaces                         |     |     |
| 4. Rubs hands together vigorously for minimum 20 seconds           |     |     |
| 5. Cleans between fingers, backs of hands, thumbs, and under nails |     |     |
| 6. Rinses hands thoroughly under running water, fingertips down    |     |     |
| 7. Dries hands with single-use paper towel                         |     |     |
| 8. Uses paper towel to turn off faucet                             |     |     |
| 9. Disposes of paper towel without re-contaminating hands          |     |     |

## SKILL 2: VITAL SIGNS — BLOOD PRESSURE (MANUAL)

| Step                                                              | S   | U   |
| ----------------------------------------------------------------- | --- | --- |
| 1. Performs hand hygiene                                          |     |     |
| 2. Explains procedure to patient                                  |     |     |
| 3. Positions patient with arm supported at heart level            |     |     |
| 4. Selects correct cuff size                                      |     |     |
| 5. Applies cuff to bare upper arm, 1 inch above antecubital fossa |     |     |
| 6. Aligns artery marker over brachial artery                      |     |     |
| 7. Locates brachial pulse with fingertips                         |     |     |
| 8. Places stethoscope diaphragm over brachial artery              |     |     |
| 9. Inflates cuff to 30 mmHg above point where pulse disappears    |     |     |
| 10. Deflates cuff at 2–3 mmHg per second                          |     |     |
| 11. Identifies first Korotkoff sound as systolic pressure         |     |     |
| 12. Identifies disappearance of sounds as diastolic pressure      |     |     |
| 13. Records reading accurately (e.g., 118/76 mmHg)                |     |     |
| 14. Removes cuff and performs hand hygiene                        |     |     |
| 15. Reports reading to instructor/nurse                           |     |     |

**Pass/Fail Criteria:** Steps 5, 9, 10, 11, 12, 13 are critical steps.
**Result:** ☐ PASS ☐ FAIL

## SKILL 3: VITAL SIGNS — RADIAL PULSE AND RESPIRATION

| Step                                                              | S   | U   |
| ----------------------------------------------------------------- | --- | --- |
| 1. Performs hand hygiene                                          |     |     |
| 2. Explains procedure to patient                                  |     |     |
| 3. Places index and middle fingers (not thumb) over radial artery |     |     |

| 9. Performs hand hygiene                                          |     |     |
